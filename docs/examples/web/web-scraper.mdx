## Overview

The AI Web Scraper API allows you to extract specific information from any website using advanced scraping techniques. Unlike traditional web scrapers that require complex CSS selectors or XPath expressions, our AI-powered scraper provides flexible data extraction methods.

- Extract data using multiple approaches
- No need for complex selectors, XPath, or DOM knowledge
- Works with dynamic websites and JavaScript-rendered content
- Handles various data structures (text, lists, tables)
- Adapts to changing website layouts

## API Endpoint

```
POST /ai/scrape
```

## Quick Start

```javascript
import { JigsawStack } from "jigsawstack";

const jigsaw = JigsawStack({ apiKey: "your-api-key" });

const response = await jigsaw.scrape({
  url: "https://example.com",
  selectors: ["h1", "meta[name='description']"],
  element_prompts: [
    "main title",
    "page description"
  ]
});

console.log(response);
```

## Request Parameters

- `url` (string, optional): The website URL to scrape
- `html` (string, optional): Direct HTML content to parse
- `selectors` (array, optional): CSS selectors to extract specific elements
- `element_prompts` (array, optional): Natural language prompts for data extraction
- `wait_for` (object, optional): Configuration to wait for specific page conditions
- `goto_options` (object, optional): Page navigation and loading configurations

## Response Example

```json
{
  "success": true,
  "data": {
    "main_title": "Product Page",
    "description": "Detailed product information",
    "extracted_details": {
      "price": "$99.99",
      "availability": "In Stock"
    }
  },
  "page_position": 1,
  "status_code": 200
}
```

## Advanced Usage

### Dynamic Content Handling

```javascript
const response = await jigsaw.scrape({
  url: "https://dynamic-website.com",
  wait_for: {
    mode: "selector",
    value: ".content-loaded"
  },
  goto_options: {
    timeout: 30000,
    wait_until: "networkidle2"
  },
  element_prompts: [
    "Load all product details",
    "Extract pricing information"
  ]
});
```

## Writing Effective Prompts

1. **Be specific**: Use precise language to guide extraction
2. **Provide context**: Clarify what information you need
3. **Specify format**: Indicate desired output structure
4. **Be descriptive**: Help the AI understand your intent

## Error Handling

The API returns a structured response with:
- `success`: Boolean indicating request status
- `data`: Extracted information
- `error`: Detailed error message if extraction fails

### Possible Error Scenarios
- Invalid URL
- Network connectivity issues
- Page loading timeout
- Complex page structures
