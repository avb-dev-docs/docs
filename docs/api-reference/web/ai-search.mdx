## Overview

The AI Search API allows you to perform web searches with AI-enhanced results. It provides options for customizing the search experience, including AI-powered overviews, safe search settings, and deep research capabilities.

## Request Parameters

### Body

<ParamField body="query" type="string" required>
  The search query. Maximum character length is 400.
</ParamField>

<ParamField body="ai_overview" type="boolean" default="true">
  Include AI-powered overview in the search results.
</ParamField>

<ParamField body="safe_search" type="string" default="moderate">
  Control the level of safe search filtering:
  <ul>
    <li>`moderate` - Default setting</li>
    <li>`strict` - More aggressive filtering</li>
    <li>`off` - No content filtering</li>
  </ul>
</ParamField>

<ParamField body="spell_check" type="boolean" default="true">
  Enable spell checking for the search query.
</ParamField>

<ParamField body="byo_urls" type="string[]">
  Specify URLs to include in the search results.
</ParamField>

<ParamField body="country_code" type="string">
  Two-letter country code to localize search results.
</ParamField>

<ParamField body="auto_scrape" type="boolean">
  Enable automatic scraping of search results.
</ParamField>

<ParamField body="deep_research" type="boolean">
  Enable deep research mode for more comprehensive results.
</ParamField>

<ParamField body="deep_research_config" type="object">
  Configure deep research parameters:
  <Expandable title="Deep Research Config Object">
    <ParamField body="max_depth" type="number" default="3">
      Number of recursive iterations for deep research.
    </ParamField>
    <ParamField body="max_breadth" type="number" default="3">
      Number of parallel subqueries generated at each depth level.
    </ParamField>
    <ParamField body="max_output_tokens" type="number" default="32000">
      Maximum number of tokens in the generated output report.
    </ParamField>
    <ParamField body="target_output_tokens" type="number">
      Target number of tokens for the generated output report.
    </ParamField>
  </Expandable>
</ParamField>

## Usage Examples

### Basic Search

```javascript
const response = await jigsaw.web.search({
  query: "AI in healthcare",
  ai_overview: true,
  safe_search: "moderate"
});
```

### Deep Research

```javascript
const response = await jigsaw.web.search({
  query: "Impact of climate change on biodiversity",
  deep_research: true,
  deep_research_config: {
    max_depth: 4,
    max_breadth: 5,
    max_output_tokens: 50000
  }
});
```

### Custom URL Search

```javascript
const response = await jigsaw.web.search({
  query: "Latest AI research",
  byo_urls: [
    "https://arxiv.org/list/cs.AI/recent",
    "https://www.nature.com/subjects/artificial-intelligence"
  ]
});
```

## Best Practices

1. Use specific and well-formed queries for better results.
2. Enable `ai_overview` for a quick summary of search results.
3. Adjust `safe_search` settings based on your application's requirements.
4. Utilize `deep_research` for comprehensive analysis on complex topics.
5. Use `byo_urls` to focus searches on specific trusted sources.
6. Consider using `country_code` for location-specific results when relevant.
