## Overview

The AI Search API allows you to perform web searches with AI-enhanced results. It provides options for customizing the search experience, including safe search settings, AI-generated overviews, and deep research capabilities.

## Request Parameters

### Body

<ParamField body="query" type="string" required>
  The search query. Maximum length is 400 characters.
</ParamField>

<ParamField body="ai_overview" type="boolean" default="true">
  Include an AI-powered overview in the search results.
</ParamField>

<ParamField body="safe_search" type="string" default="moderate">
  Control the inclusion of potentially offensive results:
  <ul>
    <li>`moderate` - Default setting, filters most explicit content</li>
    <li>`strict` - Applies the highest level of filtering</li>
    <li>`off` - No content filtering applied</li>
  </ul>
</ParamField>

<ParamField body="spell_check" type="boolean" default="true">
  Perform spell checking on the search query.
</ParamField>

<ParamField body="byo_urls" type="string[]">
  Array of specific URLs to include in the search results.
</ParamField>

<ParamField body="deep_research" type="boolean">
  Enable deep research mode for more comprehensive results.
</ParamField>

<ParamField body="deep_research_config" type="object">
  Configuration options for deep research mode.
  <Expandable title="Deep Research Config Object">
    <ParamField body="max_depth" type="number" default="3">
      Number of recursive iterations for research. Higher values enable more thorough but longer searches.
    </ParamField>
    <ParamField body="max_breadth" type="number" default="3">
      Number of parallel subqueries generated at each depth level. Higher values provide wider coverage.
    </ParamField>
    <ParamField body="max_output_tokens" type="number" default="32000">
      Maximum number of tokens in the generated output report.
    </ParamField>
    <ParamField body="target_output_tokens" type="number">
      Target number of tokens for the generated output report.
    </ParamField>
  </Expandable>
</ParamField>

## Usage Examples

```javascript
import { JigsawStack } from "jigsawstack";

const jigsaw = JigsawStack({ apiKey: "your-api-key" });

const response = await jigsaw.web.search({
  query: "Best restaurants in New York",
  ai_overview: true,
  safe_search: "moderate",
  deep_research: true,
  deep_research_config: {
    max_depth: 4,
    max_breadth: 5
  }
});

console.log(response);
```

```python
from jigsawstack import JigsawStack

jigsaw = JigsawStack(api_key="your-api-key")

response = jigsaw.web.search(
    query="Best restaurants in New York",
    ai_overview=True,
    safe_search="moderate",
    deep_research=True,
    deep_research_config={
        "max_depth": 4,
        "max_breadth": 5
    }
)

print(response)
```

## Best Practices

1. Use specific and concise search queries for better results.
2. Enable `ai_overview` for a quick summary of the search results.
3. Adjust `safe_search` settings based on your application's requirements.
4. Utilize `deep_research` for more comprehensive results on complex topics.
5. Configure `deep_research_config` parameters to balance between search depth and response time.
