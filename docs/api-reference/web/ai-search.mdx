## Deep Research Mode

When `deep_research` is set to `true`, the AI Search API performs a more comprehensive analysis of the search topic. This mode is particularly useful for complex queries that require in-depth exploration of multiple related subtopics.

The `deep_research_config` object allows you to fine-tune the behavior of the deep research mode:

- `max_depth`: Controls how many levels deep the research goes. Higher values result in more comprehensive but potentially longer searches.
- `max_breadth`: Determines how many parallel subtopics are explored at each depth level. Higher values provide wider coverage but may increase processing time.
- `max_output_tokens`: Sets an upper limit on the length of the generated report.
- `target_output_tokens`: Provides a target length for the output, which the API will attempt to meet without exceeding `max_output_tokens`.

When using deep research mode, expect longer processing times but more comprehensive and interconnected results.
