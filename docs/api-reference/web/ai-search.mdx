## Overview

The AI Search API allows you to perform web searches and retrieve high-quality results enhanced by artificial intelligence. This API provides features like AI-powered overviews, safe search options, and deep research capabilities.

## Request Parameters

### Body

<ParamField body="query" type="string" required>
  The search query. Maximum length is 400 characters.
</ParamField>

<ParamField body="ai_overview" type="boolean" default="true">
  Include an AI-powered overview in the search results.
</ParamField>

<ParamField body="safe_search" type="string" default="moderate">
  Control the inclusion of offensive results:
  <ul>
    <li>`moderate` (default)</li>
    <li>`strict`</li>
    <li>`off`</li>
  </ul>
</ParamField>

<ParamField body="spell_check" type="boolean" default="true">
  Perform spell checking on the search query.
</ParamField>

<ParamField body="byo_urls" type="string[]">
  Specify URLs to include in the search results.
</ParamField>

<ParamField body="country_code" type="string">
  Two-letter country code to localize search results.
</ParamField>

<ParamField body="auto_scrape" type="boolean">
  Enable automatic scraping of search results.
</ParamField>

<ParamField body="deep_research" type="boolean">
  Enable deep research mode for more comprehensive results.
</ParamField>

<ParamField body="deep_research_config" type="object">
  Configuration for deep research mode:
  <Expandable title="Deep Research Config Object">
    <ParamField body="max_depth" type="number" default="3">
      Number of recursive iterations for research depth.
    </ParamField>
    <ParamField body="max_breadth" type="number" default="3">
      Maximum number of parallel subqueries at each depth level.
    </ParamField>
    <ParamField body="max_output_tokens" type="number" default="32000">
      Maximum number of tokens in the generated output report.
    </ParamField>
    <ParamField body="target_output_tokens" type="number">
      Target number of tokens for the generated output report.
    </ParamField>
  </Expandable>
</ParamField>

## Usage Examples

Here's an example of how to use the AI Search API:

```javascript
import { JigsawStack } from "jigsawstack";

const jigsaw = JigsawStack({ apiKey: "your-api-key" });

const response = await jigsaw.web.search({
  query: "latest advancements in artificial intelligence",
  ai_overview: true,
  safe_search: "moderate",
  deep_research: true,
  deep_research_config: {
    max_depth: 4,
    max_breadth: 5,
  }
});

console.log(response);
```

This example performs a search on the latest advancements in AI, includes an AI-powered overview, uses moderate safe search, and enables deep research for more comprehensive results.
